# portfolio
I did an example using ReactJS and NodeJS.
I used some frameworks like:
* ChartJS for Charts
* Socket.IO to REAL-TIME ENGINE


# Install API Server

* Install and start the API server
    - `cd api-server`
    - `npm install`
    - `node server`
    
# Install React Application

* Install and start the React Application
    - `cd front-end`
    - `npm install`
    - `npm start`
